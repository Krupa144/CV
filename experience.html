PHP

During my internship at Omega Pilzno ITiS Godawski & Godawski Sp. z o.o., my primary responsibility was architecting and delivering a full-stack web application that tracks company carts in real time. Each cart receives a unique QR code upon creation; warehouse staff scan the code with a mobile device, and the system instantly records its current location, usage status, and maintenance history. The back end, built with PHP8 and SQL, is secured with JWT, while the front end (JavaScript + jQuery) renders live dashboards and push notifications via WebSockets.

Beyond the core tracking functionality, I implemented an inventory module that allows managers to define storage zones, set stock thresholds, and schedule automated alerts when inventory falls below set limits. For reporting, I developed an export engine that compiles daily or monthly summaries — complete with pivot tables and conditional formatting — directly into Excel (.xlsx) files, ready for finance or audit teams. This automation significantly reduced manual spreadsheet work.
